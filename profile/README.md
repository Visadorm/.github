## 🌍 What Is Visadorm?

**Visadorm is a non-custodial travel settlement protocol.**

We are building decentralized infrastructure that enables global travel bookings to settle in USDC — without platform custody risk.

✔ No custody of user funds
✔ No governance token
✔ No emissions
✔ No speculative mechanics

Infrastructure — not marketplace.

## 🚀 Designed for 1 Trillion+ Bookings Per Year

Visadorm is architected for extreme global scale.

Infrastructure assumptions:

* Massive global booking throughput
* Continuous USDC settlement
* High-frequency escrow execution
* Long-term gas efficiency

Smart contracts are optimized for:

* Minimal storage footprint
* Constant gas complexity (O(1))
* Efficient escrow logic
* Stateless verification where possible

Planet-scale settlement layer.

## 💼 Non-Custodial Wallet Layer

✅ Non-custodial wallet interface
✅ Store USDC
✅ Send / receive USDC
✅ Accept USDC via on-chain wallet
✅ Refunds / cancellations
✅ Disputes
✅ No centralized custody

All settlements occur directly on-chain.

---

## 🔐 Core Flow

1️⃣ Guest deposits USDC into escrow
2️⃣ Funds locked in smart contract
3️⃣ Service completed
4️⃣ Funds released to host
5️⃣ Disputes resolved via protocol-controlled wallet

No off-chain custody.
No centralized holding accounts.

---

## ⛽ Gas Strategy

* Platform-sponsored gas (guest never needs ETH)
* Gas abstraction via relayer model
* Batch settlement support
* Built-in gas volatility hedge inside protocol fee

Gas is treated as infrastructure overhead — not user friction.
